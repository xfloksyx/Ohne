  • Antivirus: Some antivirus software may flag the installer. This is a false positive due to the AI model downloads.
  • Dependencies: All required dependencies are included in the installer.
  • Updates: The application will automatically check for updates on startup.
  • Uninstall: Use Windows "Add or Remove Programs" to uninstall cleanly.

Quick Start Guide

Processing a YouTube Video

  1. Launch Ohne - Only Vocals
  2. Navigate to Main Tab
  3. Enter YouTube URL in the input field
  4. Set your desired output filename
  5. Choose processing mode (Extract Vocals or Merge with Video)
  6. Click "Start Processing" and monitor progress

Processing a Local File

  1. Click "Select Local File" button
  2. Choose your video file (MP4, MOV, MKV, WEBM)
  3. Set output filename
  4. Select processing mode
  5. Start processing and wait for completion
  6. Find your processed file in the videos folder

Processing Modes Explained

Extract Vocals Only (WAV)

  • Output: High-quality WAV audio file
  • Use Case: Karaoke tracks, vocal analysis, remixing
  • Quality: 44.1kHz/16-bit or higher
  • File Size: 10-50 MB for 3-4 minute song

Merge with Video (MP4)

  • Output: MP4 video with extracted vocals
  • Use Case: Vocal-only music videos
  • Quality: Original video resolution maintained
  • File Size: Similar to original video

Troubleshooting

"Failed to download video"

Cause: Network connectivity or invalid URL

Solution:

  • Check internet connection
  • Verify YouTube URL is correct
  • Ensure video is publicly available
  • Try again after a few minutes

"Processing failed during extraction"

Cause: Insufficient memory or corrupted audio

Solution:

  • Close other applications to free memory
  • Try processing shorter video segments
  • Restart the application
  • Check available disk space

Slow Processing Speed

Optimization Tips:

  • Close unnecessary applications
  • Use SSD storage for better performance
  • Ensure adequate RAM availability
  • Process shorter video segments

Technical Specifications

Audio Processing

  • AI Model: Demucs v4
  • Sample Rate: Up to 48kHz
  • Bit Depth: 16-bit and 24-bit
  • Channels: Stereo with mono compatibility

Video Processing

  • Input: MP4, MOV, MKV, WEBM, AVI
  • Output: MP4 (H.264), WAV (PCM)
  • Resolution: Up to 4K (3840x2160)
  • Frame Rates: 24fps, 30fps, 60fps

Network Features

  • YouTube: Full URL compatibility
  • Quality: Automatic best selection
  • Bandwidth: Optimized streaming
  • Caching: Efficient temporary storage
